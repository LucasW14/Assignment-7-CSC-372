## Setup Instructions

1. Clone the repository:
   git clone https://github.com/yourusername/rps-game.git

2. Navigate into the folder:
   cd rps-game

3. Install dependencies:
   npm install

4. Start the app:
   npm run dev


Reflection on implementation 

In this project I made a Rock, Paper, Scissors game using react. I used useState to manage the Player's choice, PLayer's score, Opponent Choice, and Opponent Score, images, and winner. One challenge I faced was finding a way to make the image shuffle to a new picture every 500 ms for 3 seconds. Iw as able to fix this problem using the "const intervalRef = useRef(null)" concept.I used one main component which was the RPS component then I had small functions inside that one.
